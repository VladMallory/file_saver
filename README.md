# Установка
## Установщик с интерфейсом
```bash
curl -s https://raw.githubusercontent.com/VladMallory/file_saver/main/install.sh | bash
```
Дальше только если хотите вручную 

---
## Ручная настройка
### Заполнить файлы. `.env` и `path.txt`
В папке `save-file` нужно создать `.env` и там прописать переменные с своими параметрами
```.env
TELEGRAM_TOKEN=
TELEGRAM_CHAT_ID=
```

```bash
cd save-file
nano .env
```

- `TELEGRAM_TOKEN` - берется из @BotFather. Нужно будет создать бота, или если уже есть, взять токен из существующего
- `TELEGRAM_CHAT_ID` - берется из @Getmyid_bot. **Вроде бы не требует подписку и не показывает рекламу**


### Ручной запуск
В таком варианте не будет cron (запуск каждый день в определенное время). Он просто запустит принудительно, если хотите запуск каждый день, то нужно запускать через установщик
```bash
./save-file/file_saver run
```
